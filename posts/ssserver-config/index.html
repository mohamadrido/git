Shadowsocks Server 配置

Cotes Chung on Jun 25, 20182018-06-25T14:18:00+08:00
1 min read views

配置

服务端采用 JSON 文件存储配置信息，可分单用户和多用户两种情况：

单用户

1
2
3
4
5
6
7

{
  "server": "0.0.0.0",
  "server_port": 10000,
  "password": "mypasswd",
  "timeout": 300,
  "method": "aes-256-cfb"
}

多用户

1
2
3
4
5
6
7
8
9

{
  "server": "0.0.0.0",
  "port_password": {
    "1001": "password_a",
    "1002": "password_b"
  },
  "timeout": 300,
  "method": "aes-256-cfb"
}

启动命令

1

$ /bin/ssserver -c /path/to/config.json -d start

参数说明：

-c 表示加载指定配置文件

-d 表示后台模式
